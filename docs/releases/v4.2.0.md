# v4.2.0

发布日期：2026-05-05

## 升级摘要

v4.2.0 是一次结构整治版本。原 7 项 hot-spot 全部收敛到 600 行 CLAUDE.md 软上限以下；
3 个 endpoint 层 god-module（analysis / infrastructure / macro_quality）按子主题
拆为多个 sub-router 包；4 个 package `__init__` 的死 re-export 全部清理；同步
完成多轮死代码扫荡（废 strategy/util 模块、orphan 端点 + 配套测试与 service
helper、frontend orphan 目录、scratch 脚本）。

无新增功能、无 API 契约变化（OpenAPI 路径与字段全程 zero-drift）；这是一次
纯结构与卫生升级。
